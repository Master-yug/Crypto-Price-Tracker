# PixelCoin Tracker v2.0
A retro-futuristic, cyberpunk-themed cryptocurrency tracker built with vanilla HTML, CSS, and JavaScript. This dashboard provides real-time market data with a nostalgic 8-bit arcade aesthetic.
## Features
* **Retro UI:** CRT scanline overlays, neon glowing text, and pixel-art typography.
* **Live Market Data:** Fetches top 50 cryptocurrencies via the CoinGecko API.
* **Zero Dependencies:** No frameworks (React/Vue) or libraries (Bootstrap/Chart.js)â€”just pure code.
* **SVG Charting:** Custom-written SVG generator for 7-day price sparklines.
* **Search & Sort:** Filter assets instantly and sort by Price, Volume, or Market Cap.
* **Responsive:** optimized for mobile, tablet, and desktop screens.
